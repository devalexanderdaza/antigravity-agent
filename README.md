## 📋 操作指南

### 第一次使用
1. 启动 Antigravity Agent
2. 点击"刷新"按钮检测当前账户
3. 系统会自动保存当前登录的用户

### 刷新按钮
- 检测当前登录的账户
- 自动备份当前账户的凭证
- 显示所有已保存的账户列表

### 切换账户
- 从账户列表中选择要切换的账户
- 点击"切换"按钮自动切换

### 登录新账户
- 点击"登录新账户"按钮
- 确认操作后将清除当前用户数据
- 自动启动 Antigravity，登录账户点击刷新即可保存凭证

### 导入导出
- **导出**: 选择保存位置，设置密码，导出所有账户配置
- **导入**: 选择配置文件，输入密码，恢复账户数据
